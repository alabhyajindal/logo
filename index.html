The Logo Programming Langauge

Logo is a visual programming language. Type commands into the input and experiment. Draw shapes, have fun.

rt <degree>

rotate right

lt <degree>

rotate left

fd <steps>

move forward

bk <steps>

move backward

repeat <count> [commands]

repeat commands

pc <number>

change pen color

pd

put pen down

pu

pick pen up

cs

clear screen

ct

center turtle
