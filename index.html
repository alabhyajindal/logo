The Logo Programming Langauge

Logo is a visual programming language. Type commands into the input and experiment. Draw shapes, have fun.

fd <steps>

move forward

bk <steps>

move backward

rt <degree>

rotate right

lt <degree>

rotate left

pd

put pen down

pu

pick pen up

ct

center turtle

cs

clear screen

pc <number>

change pen color

repeat <count> [commands]

repeat commands
